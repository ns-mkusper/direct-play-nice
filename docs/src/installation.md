# Installation

## Build from source

```bash
git clone https://github.com/ns-mkusper/direct-play-nice.git
cd direct-play-nice
cargo build --release
```

The resulting binary is at:

```text
target/release/direct_play_nice
```

## Verify installation

```bash
./target/release/direct_play_nice --version
./target/release/direct_play_nice --help
```
